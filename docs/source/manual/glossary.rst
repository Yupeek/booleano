********
Glossary
********

.. glossary::

    adaptive grammar
        A :term:`grammar` whose properties can be overridden.
        
        .. seealso::
            `Adaptive grammar <http://en.wikipedia.org/wiki/Adaptive_grammar>`_
            at the Wikipedia.
    
    binding
        http://en.wikipedia.org/wiki/Name_binding
    
    grammar
        doo
    
    literal
        An :term:`operand` whose value is used explicitly in a boolean
        expression. In Booleano, it can be an :term:`string <string literal>`,
        a :term:`number <number literal>` or a :term:`set <set literal>`.
        
        .. seealso::
        
            `Literal <http://en.wikipedia.org/wiki/Literal_(computer_science)>`_
            at the Wikipedia.
    
    namespace
        sdqw
    
    number literal
        The :term:`literal` representation of a number.
    
    operand
        In Booleano,
    
    
    set literal
        The :term:`literal` representation of set of elements. The elements can
        be any :term:`operand`, even another set literal.
    
    
    string literal
        The :term:`literal` representation of an string.
        
        In the default :term:`grammar`, strings are delimited by single or
        double quotes, so the string ``Hello world`` can be written as:
        
        * ``"Hello world"``
        * ``'Hello world'``
        
        .. seealso::
        
            `String literals <http://en.wikipedia.org/wiki/String_literal>`_ at
            the Wikipedia.
